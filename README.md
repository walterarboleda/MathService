# MathService

# Requisitos Previos Visual Studio (2022 o superior).

# Creación del Servidor (Web Service ASMX)
1. Abre Visual Studio y crea un nuevo proyecto de tipo ASP.NET Web Application (.NET Framework) en C#.

2. Selecciona la plantilla Empty (Vacía).

3. Haz clic derecho en el proyecto -> Add -> New Item -> Web Service (ASMX). Ponle de nombre MathService.asmx.

4. Digital o pega el Código del Servidor (MathService.asmx.cs)

5. Ejecuta el proyecto (F5). Se abrirá un navegador con la dirección (ej. http://localhost:54321/MathService.asmx). Copia esta URL, la necesitarás para el cliente.
